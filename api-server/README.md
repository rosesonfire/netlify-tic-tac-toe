# API Server
GraphQL server for back end

# Production

Set `ENV=production` in [.env](../.env).


## URL

http://localhost:4000

# Development

Set `ENV=development` in [.env](../.env).


## URL

http://localhost:4001

# Commands

## Run service

```bash
../start api-server
```

## Restart service

```bash
../restart api-server
```

## Stop service

```bash
../stop api-server
```

## View logs

```bash
../logs api-server
```

## Shell into service

```bash
../shell api-server
```
